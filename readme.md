# MagnopVS_Web

# 🌌 MagnopVS Web

> **La puerta de entrada al Quantvm Matrix.**

MagnopVS Web es la aplicación web oficial del universo transmedia **MagnopVS**, desarrollada por **Nenca Lab**.

No es una página web tradicional. Es una experiencia interactiva que conecta narrativa, videojuegos, exploración, comunidad y crecimiento personal dentro del ecosistema MagnopVS.

---

# ✨ Objetivos

- Crear una experiencia inmersiva desde el primer segundo.
- Presentar el universo de MagnopVS mediante un onboarding interactivo.
- Integrar el Quantvm Matrix con el Atlas Cósmico.
- Conectar el sitio con Supabase para usuarios, progreso y contenido.
- Servir como núcleo de toda la plataforma transmedia.

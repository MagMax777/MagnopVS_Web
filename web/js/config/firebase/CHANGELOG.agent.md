# [0.1.0] - 2026-07-17

## 🎉 Inicio del Proyecto

### Agregado

- Creación del repositorio principal.
- Organización inicial del proyecto.
- Inicio del desarrollo del sitio web.
- Definición de la arquitectura general.

### Documentación

- Creación de README.md
- Creación de DESIGN.md
- Creación de ROADMAP.md
- Creación de API.md
- Creación de MAGNOPVS_DATABASE_ARCHITECTURE.md
- Creación de CHANGELOG.md

### Frontend

- Diseño inicial del Home.
- Diseño inicial del Marketplace.
- Diseño inicial del Hero Game.
- Diseño inicial del Register.
- Diseño del sistema visual AAA.

### Backend

- Configuración inicial de Firebase Authentication.
- Configuración del proyecto Supabase.
- Definición de utilizar PostgreSQL como base de datos principal.
- Definición de utilizar Firebase únicamente para autenticación.

### Arquitectura

- Separación de responsabilidades entre Firebase y Supabase.
- Arquitectura modular del backend.
- Definición de módulos del sistema.

### Próximos pasos

- Sistema de Login.
- Conexión Firebase.
- Conexión Supabase.
- Creación de Profiles.
- Dashboard.
